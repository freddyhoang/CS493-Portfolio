API info here:

Following endpoints for API usage:

Post + get allowed for the following endpoints:

  • /dates
  • /meals
  • /foods

Delete + get + put allowed for the following endpoints:

  • /dates/dates_id
  • /meals/meals_id
  • /foods/foods_id

Patch + delete allowed for the following endpoints:

  • /dates/dates_id/meals_id
  • /meals/meals_id/foods_id

Get allowed for the following endpoints:

  • /dates/dates_id/meals
  • /meals/meals_id/dates
  • /meals/meals_id/foods
  • /foods/foods_id/meals
  • /users
